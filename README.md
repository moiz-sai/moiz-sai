## 🚀 Featured Project

### 🏗️ [AI Building Energy Prediction](https://github.com/moiz-sai/AI-Building-Energy-Prediction)

> Predicting building heating load from architectural features using ML regression models.

- **Dataset:** UCI ENB2012 — 768 samples, 8 features
- **Models:** Linear Regression · Polynomial Regression · Decision Tree Regressor
- **Best result:** R² = 0.99, RMSE = 1.02 (Decision Tree)
- **Stack:** Python · scikit-learn · Pandas · Matplotlib · Tkinter GUI
